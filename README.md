# Tauri + Sveltekit + Typescript template

This template contains:
- Tauri
- Sveltekit
- Typescript
- Tailwind
- Eslint
- Prettier
- Playwright

## Creating a project
```bash
npx degit Inv1x/tauri-sveltekit-typescript-template <project-name>
cd <project-name>
yarn
```

## Developing
To run development SvelteKit server
```bash
yarn dev
```

To run development Tauri app
```bash
yarn tauri dev
```
